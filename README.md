# eepy.page

eepy.page offers subdomains to anyone, free of charge!

## available domains
- .eepy.page
- .worksonmymachine.top
- .opticalmediagood.top

## languages

i have removed internationalization to assist with the migration of this project. as this is quite a small project, i'm going to keep it in english for now, but if it grows, i'll consider adding translations/taking contributions for translations.
